## 如何添加友链

添加本站链接后，评论/向邮箱`test19124@sina.com` 发送如下内容：

```json
{
  url: "",    #站点网址
  title: "",  #站点名称
  desc: "",   #简短描述
  author: "", #管理员名字
  avatar: "", #一张图片
},
```

## 友链要求

欢迎交换友链！请在提交前确认以下事项：

1. 网站不能存在违反中国法律的内容
2. 网站不能是纯广告性质 (本人主观判断) 的网站
3. 网站不能存在大量令人不适的内容
4. 不能是盗版资源 (或有争议的资源) 为主的资源站
